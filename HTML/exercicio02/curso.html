React

Sobre o curso

React é uma biblioteca JavaScript de código aberto para criar interfaces de usuário, mantida pelo Facebook.

É usada principalmente para construir single-page applications (SPAs) e interfaces móveis.

Seu principal recurso é a capacidade de criar componentes reutilizáveis, tornando o desenvolvimento mais eficiente e organizado.
